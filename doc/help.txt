CPTAC help:

get functions:
  get_clinial():
    Returns clinical.csv as a data frame.
  get_proteomics():
    Returns proteomics.txt as a data frame.
  get_transcriptome():
    Returns transcriptome.txt as a data frame.
  get_cna():
    Returns CNA.txt as a data frame.
  get_phosphoproteomics():
    Returns phosphoproteomics.txt as a data frame.
  get_meta_cols():
    Returns column names of clinical data frame.
  get_cohort_meta(cols):
    Returns specified columns of clinical data frame.
  get_proteomics_quant(colon_ids):
    Returns specified indices of proteomics data frame.
  get_proteomics_cols():
    Returns column names of proteomics data frame.
  get_transcriptome_cols():
    Returns column names of transcriptome data frame.
  get_cohort_proteomics(cols):
    Returns specified columns of proteomics data frame.
  get_cohort_transcriptome(cols):
    Returns specified columns of transcriptome data frame.
  get_cohort_cna(cols):
    Returns specified columns of CNA data frame.
  get_cohort_phosphoproteomics(cols):
    Returns specified columns of phosphoproteomics data frame.

Utilities:
  get_gene_mapping():
    Returns gene mapping.
  convert(snp_or_sap):
    Converts provided SNP to SAP or vice versa.
  compare_gene(df1, df2, gene):
    Selects for specified gene from two provided data frames. Returns data frame
    with two columns for intersection of indices between both data frames.
  compare_clinical(clinical, data, clinical_col):
    Selects specified clinical column and adds it to the specified data, i.e.
    proteomics, CNA etc., returning a data frame of the combined data.

Queries:
  query(tumor_type, query_type, value):
    Queries data based on 'query_type', such as by 'SNP',
    'mutated gene', or 'outlier'. Query is based on 'tumor_type', such as colon.
    Value correlates with 'query_type', such as TP53 if query
    type is 'mutated gene', or 'EGFR' for 'outlier'.
