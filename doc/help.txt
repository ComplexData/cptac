CPTAC help:

Presented here is the docstring summary for all functions available in this package.

  list_data():
    Lists the attributes of accessible data in the package

get functions:
  get_clinical(excluded):
    Returns clinical.txt (optional excluded cases in clinical.txt, which includes
    all "Yes" samples under Case_excluded) as a data frame.
  get_proteomics():
    Returns proteomics.cct as a data frame.
  get_transcriptomics(circular, miRNA):
    Returns transcriptomics_linear.cct (optional transcriptomics_circular.cct
    or miRNA.cct) as a data frame.
  get_cna():
    Returns CNA.cct as a data frame.
  get_phosphoproteomics(gene_level):
    Returns phosphoproteomics_site.cct (optional phosphoproteomics_gene.cct)
    as a data frame.
  get_phosphosites(gene):
    Returns dataframe with all phosphosites of specified gene name
  get_mutations(binary, unparsed):
    Returns parsed somatic.maf (optional somatic.cbt or unparsed somatic.maf) as a data frame.
  get_clinical_cols():
    Returns column names of clinical data frame.
  get_cohort_clinical(columns):
    Returns specified columns of clinical data frame.
  get_proteomics_quant(colon_ids):
    Returns specified indices of proteomics data frame.
  get_proteomics_cols():
    Returns column names of proteomics data frame.
  get_transcriptomics_cols():
    Returns column names of transcriptomics data frame.
  get_cohort_proteomics(columns):
    Returns specified columns of proteomics data frame.
  get_cohort_transcriptomics(columns):
    Returns specified columns of transcriptomics data frame.
  get_cohort_cna(columns):
    Returns specified columns of CNA data frame.
  get_cohort_phosphoproteomics(columns):
    Returns specified columns of phosphoproteomics data frame.
  get_patient_mutations(patient_id):
    Returns data frame of mutation data matching specified patient ID.

Utilities:
  compare_omics(omics_df1, omics_df2, cols1=None, cols2=None):
    Take specified column(s) from one omics dataframe, and merge with specified columns(s) from another omics dataframe.
  append_clinical_to_omics(clinical_cols, omics_df, omics_cols=None):
    Append columns from clinical dataframe to part or all of an omics dataframe.
  append_derived_molecular_to_omics(derived_molecular_cols, omics_df, omics_cols=None):
    Append columns from derived_molecular dataframe to all or part of an omics dataframe.
  append_mutations_to_omics(mutation_genes, omics_df, omics_genes=None, multiple_mutations=False, show_location=True):
    Select all mutations for specified gene(s), and append to all or part of the given omics dataframe.
